# Open-Jarvis · Public Users Registry

> 🇮🇹 Registro pubblico degli utilizzatori di Open-Jarvis · 🇬🇧 Public registry of Open-Jarvis users

Open-Jarvis è **libero, gratuito e perpetuamente open source** sotto licenza **AGPL-3.0**.
Open-Jarvis is **free, libre and permanently open source** under the **AGPL-3.0 License**.

In conformità allo spirito del progetto e ai requisiti AGPL, **chiunque utilizzi Open-Jarvis (in qualsiasi forma) è tenuto a registrarsi pubblicamente in questo file**, e **chi modifica il software deve rendere pubbliche le proprie modifiche**.

In line with the project spirit and AGPL requirements, **anyone using Open-Jarvis (in any form) is required to register publicly in this file**, and **anyone who modifies the software must publish their modifications**.

---

## 📜 Le tre regole · The three rules

1. **Sempre gratis · Always free** — nessun fork può rendere a pagamento il core / no fork can paywall the core
2. **Sempre open source · Always open source** — qualsiasi modifica deve essere pubblicata sotto AGPL-3.0 / any modification must be released under AGPL-3.0
3. **Sempre dichiarato · Always declared** — chi usa Jarvis (specialmente in produzione, in azienda, in service-as-a-software) deve aggiungersi a questo file via PR

---

## ✍️ Come registrarsi · How to register

Apri una **Pull Request** che aggiunge una riga alla sezione corrispondente. Il maintainer la mergia entro 7 giorni.

Open a **Pull Request** adding a row to the matching section. Maintainer will merge it within 7 days.

Schema della voce / entry schema:

```markdown
- **Nome / Name** — *contesto d'uso (Personal, Family, Education, Enterprise, etc.)* — [link opzionale al sito o repo del fork] — *città, paese*
```

### Esempi · Examples

```markdown
- **Federico Calò** — *Personal* — [federicocalo.dev](https://federicocalo.dev) — *Lecce, Italia*
- **Acme Corp** — *Enterprise (200 employees)* — [acme.example](https://acme.example) — *Milano, Italia*
- **Liceo Galilei** — *Education / K-12* — *Roma, Italia*
- **Studio Medico Rossi** — *Healthcare (small practice)* — *Bari, Italia*
```

### Regole sulla privacy · Privacy rules

- Per **uso personale**, nome e città sono sufficienti (anche pseudonimo se preferisci).
  For **personal use**, name and city are enough (pseudonym is fine).
- Per **uso aziendale / institutional**, è richiesto il nome legale dell'organizzazione e una URL pubblica.
  For **business / institutional use**, legal name of the organisation and a public URL are required.
- Mai dati personali sensibili (email private, telefoni, indirizzi).
  Never share sensitive personal data (private emails, phones, addresses).

## 🤝 Spirit of the project

Open-Jarvis non è un prodotto commerciale travestito da open source. È un **bene comune digitale**: un'infrastruttura AI personale che vogliamo rimanga di tutti, sempre.

Open-Jarvis is not a commercial product disguised as open source. It is a **digital common**: a personal AI infrastructure that we want to remain everyone's, forever.
