# Agents · Agenti device-side

🇮🇹 Agenti che vivono **sui dispositivi** dell'utente e dialogano con il server centrale.
🇬🇧 Agents that live **on the user's devices** and talk to the central server.

## Protocollo · Protocol

🇮🇹 Tutti gli agenti parlano con il server tramite **MCP** (Model Context Protocol) e **A2A** (Agent-to-Agent), entrambi standard aperti.
🇬🇧 All agents talk to the server via **MCP** (Model Context Protocol) and **A2A** (Agent-to-Agent), both open standards.
